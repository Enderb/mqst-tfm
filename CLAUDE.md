## Building the thesis

Full build (required for correct citations and cross-references):

```bash
pdflatex -shell-escape main.tex
bibtex main
pdflatex -shell-escape main.tex
pdflatex -shell-escape main.tex
```

`-shell-escape` is mandatory because `minted` calls `pygments` to syntax-highlight code listings. Omitting it causes an immediate compile failure.

Check for errors without stopping:

```bash
pdflatex -shell-escape -interaction=nonstopmode main.tex 2>&1 | grep "^!"
```

VS Code compiles automatically via the LaTeX Workshop extension using the recipe in `.vscode/settings.json`.

## LaTeX class constraints (`quantumarticle.cls`)

- **No `\\` in `\title{}`**. The class saves the title in a single-line `\savebox` to decide font size. Inside that box `\raggedright` redefines `\\` to `\@centercr`, which calls `\par` and corrupts list state ‚Üí "Something's wrong--perhaps a missing \item". Use a long title on one line; the class wraps it automatically in the displayed parbox.
- The class blacklists `lmodern` and several font packages ‚Äî it loads them itself. Adding them again causes package conflicts.
- Abstract text and `\maketitle` are co-dependent: the class calls `\@maketitle` inside the `abstract` environment. Calling `\maketitle` explicitly before `\begin{abstract}` prevents a double call.
- Max 20 pages of main text (appendices do not count).

## Notebook conventions

All notebooks in `code/`:
- Load integrated IQ data from `./ssro_h5/SSRO_0.h5` (shape `(2, 2001, 2)`: axis 0 = prepared state, axis 1 = shots, axis 2 = [I, Q]). The first shot (`index 0`) is dropped as a potential acquisition artefact.
- Save all figures to `./figures/<name>.pdf` (or `.png`). Never save to the notebook's working directory root.
- Raw ADC trace data lives in `./ssro_h5/SSRO_raw.h5` (shape `(2, N_shots, T, 2)`).
