## LaTeX debugging

**"Missing \item" compilation error**
The error `! LaTeX Error: Something's wrong--perhaps a missing \item` at the line after `\maketitle` is caused by `\\` in the `\title{}` argument. The `quantumarticle` class saves the title in a `\savebox` (horizontal box); inside that box `\raggedright` redefines `\\` to `\@centercr`, which calls `\par` and corrupts list state. Fix: remove all `\\` from `\title{}`. The title wraps naturally in the displayed parbox.

**Isolating a compile error to a specific package**
Create a minimal test file at `/tmp/test_minimal.tex` with the `quantumarticle.cls` copied alongside it. Add packages one by one until the error reproduces. Run with `pdflatex -interaction=nonstopmode /tmp/test_minimal.tex 2>&1 | grep "^!"`.

## Analysis workflows

**IQ classifier evaluation**
Load `ssro_h5/SSRO_0.h5`, drop shot index 0, stack into `X = [X0; X1]` and `y = [0…0, 1…1]`. Train LDA, QDA, and MLP with `sklearn`. Evaluate with `confusion_matrix` → assignment fidelity `F = 1 - 0.5*(p10 + p01)`. Grid-search MLP hidden layer size over [4, 8, 16, 32] with 5-fold cross-validation. Key finding: no MLP beats LDA on single integrated IQ pairs.

**Synthetic raw trace generation**
Anchor simulation to statistics (µ₀, µ₁, Σ₀, Σ₁) extracted from `SSRO_0.h5`. Generate traces as Gaussian noise + oscillating IF envelope. Add T₁ relaxation events: at each time step, qubit flips with probability `dt/T1`. Self-consistency check: integrate synthetic traces and compare blob statistics to real data (target: within ~5%).

**1D CNN on raw traces**
Build a `torch.nn.Sequential` with `Conv1d` layers taking input shape `(batch, 2, T)` (2 = I/Q channels). Train on synthetic data from `SSRO_raw.h5`. Compare against LDA on integrated IQ. CNN can exploit time-domain structure (T₁ events) that integrated IQ discards.

## Hardware / QICK

**Phase calibration workflow**
Sweep loopback frequencies near the resonator frequency. Record `arg(mean(I + jQ))` at each point. Fit `φ(f) = φ₀ − 360·τ·f` with `scipy.optimize.least_squares`, handling 0/360° wraps. Store `phi_res = phi0 - 360*tau*f_res` and convert: `cfg["res_phase"] = soccfg.deg2reg(phi_res)`. Must re-run after every board reboot.

**Resonator spectroscopy with QICK**
Use a Python outer loop (not `RAveragerProgram`) to sweep both the DAC and ADC NCO frequencies together. `RAveragerProgram.update()` can only change DAC-side registers via the tProcessor; updating the ADC NCO requires an AXI write from the ARM core, which only a Python loop can do.

**Active reset with `condj`**
After the readout window closes, the FPGA threshold block writes a binary decision to a tProcessor register. `condj page, r_decision, '<', 1, 'SKIP_PI'` jumps to SKIP_PI (skipping the π-pulse) if the decision is 0 (qubit in |0⟩). The entire decision-to-pulse path is on the FPGA; latency is ~13 clock cycles (~34 ns at 384 MHz fabric clock).
