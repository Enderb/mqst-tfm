## LaTeX debugging

**"Missing \item" compilation error**
The error `! LaTeX Error: Something's wrong--perhaps a missing \item` at the line after `\maketitle` is caused by `\\` in the `\title{}` argument. The `quantumarticle` class saves the title in a `\savebox` (horizontal box); inside that box `\raggedright` redefines `\\` to `\@centercr`, which calls `\par` and corrupts list state. Fix: remove all `\\` from `\title{}`. The title wraps naturally in the displayed parbox.

**Abstract / maketitle coupling**
In `main.tex`, the `abstract` environment is redefined with `\renewenvironment` to avoid the class's internal `\@maketitle` call. This is because the title page is a manual `titlepage` environment (not `\maketitle`). Do not revert to `\maketitle` without also restoring the original abstract environment.

**Isolating a compile error to a specific package**
Create a minimal test file at `/tmp/test_minimal.tex` with the `quantumarticle.cls` copied alongside it. Add packages one by one until the error reproduces. Run with `pdflatex -interaction=nonstopmode /tmp/test_minimal.tex 2>&1 | grep "^!"`.

## Analysis workflows

**IQ classifier evaluation**
Load `ssro_h5/SSRO_0.h5`, drop shot index 0, stack into `X = [X0; X1]` and `y = [0…0, 1…1]`. Train LDA, QDA, and MLP with `sklearn`. Evaluate with `confusion_matrix` → assignment fidelity `F = 1 - 0.5*(p10 + p01)`.

Confirmed results: LDA = 0.906 (|0⟩: 94.0%, |1⟩: 87.3%), QDA = 0.9062. Grid-search MLP hidden layer size over [4, 8, 16, 32] with 5-fold CV gave 20 configs all within 0.898 ± 0.013 — no improvement over LDA. The LDA optimality is explained geometrically via Mahalanobis distance (~2.4) between class centroids, not solely by the equal-covariance assumption (QDA equals LDA, confirming equal covariance is a reasonable approximation but the real reason is the blob geometry).

**Synthetic raw trace generation**
Anchor simulation to statistics (µ₀, µ₁, Σ₀, Σ₁) extracted from `SSRO_0.h5`. Generate traces as Gaussian noise + oscillating IF envelope. Add T₁ relaxation events: at each time step, qubit flips with probability `dt/T1`. Self-consistency check: integrate synthetic traces and compare blob statistics to real data (target: within ~5%).

Current calibration uses T=600 samples (~30% relaxation fraction). Target is T=1200–1500 samples to match the measured ~12.75% error rate in real integrated IQ data.

**1D CNN on raw traces**
Build a `torch.nn.Sequential` with `Conv1d` layers taking input shape `(batch, 2, T)` (2 = I/Q channels). Train on synthetic data from `SSRO_raw.h5` (synthetic, pending rename to `SSRO_raw_synthetic.h5`). 953-parameter model achieves overall fidelity 0.994. Key result: on shots where the qubit relaxes mid-readout, CNN recovers 98.4% vs 44.3% for LDA — because integrated IQ averages over the T₁ flip and places the shot between the blobs, while the CNN can detect the signature in the time domain.

## Hardware / QICK

**Phase calibration workflow**
Sweep loopback frequencies near the resonator frequency. Record `arg(mean(I + jQ))` at each point. Fit `φ(f) = φ₀ − 360·τ·f` with `scipy.optimize.least_squares`, handling 0/360° wraps. Store `phi_res = phi0 - 360*tau*f_res` and convert: `cfg["res_phase"] = soccfg.deg2reg(phi_res)`. τ is the DAC/ADC pipeline latency (not cable length). Must re-run after every board reboot.

**Resonator spectroscopy with QICK**
Use a Python outer loop (not `RAveragerProgram`) to sweep both the DAC and ADC NCO frequencies together. `RAveragerProgram.update()` can only change DAC-side registers via the tProcessor; updating the ADC NCO requires an AXI write from the ARM core, which only a Python loop can do.

**Active reset with `condj`**
After the readout window closes, the FPGA threshold block writes a binary decision to a tProcessor register. `condj page, r_decision, '<', 1, 'SKIP_PI'` jumps to SKIP_PI (skipping the π-pulse) if the decision is 0 (qubit in |0⟩). The entire decision-to-pulse path is on the FPGA; latency is ~13 clock cycles (~34 ns at 384 MHz fabric clock). **Note**: the π-pulse calibration and active reset procedure are described and designed in ch. 06 but have not been demonstrated on a real qubit device within this thesis.
